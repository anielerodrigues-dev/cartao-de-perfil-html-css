# 🪪 Cartão de Perfil — HTML e CSS

Um cartão de perfil simples, desenvolvido para praticar e consolidar conceitos fundamentais de HTML e CSS.

## 📖 Sobre o projeto

Este projeto foi desenvolvido como parte dos meus estudos em HTML e CSS.

O objetivo foi praticar:

- Estrutura básica do HTML
- Seletores CSS
- Cores
- Fontes
- Margin e Padding
- Border Radius
- Box Shadow
- Hover
- Transition
- Transform

## 🛠 Tecnologias

- HTML5
- CSS3

## 🎯 Objetivo

Praticar os fundamentos do desenvolvimento Front-end e construir uma base sólida para projetos maiores.
